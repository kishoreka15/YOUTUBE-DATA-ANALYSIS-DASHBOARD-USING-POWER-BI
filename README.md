# YOUTUBE-DATA-ANALYSIS-DASHBOARD-USING-POWER-BI

## Project Overview
The YouTube Data Analysis Dashboard is a Power BI project designed to analyze and visualize YouTube channel performance data.  
It transforms raw data into meaningful insights using interactive dashboards and visualizations.  
The project helps in understanding key metrics such as views, likes, comments, and engagement rate.  
It provides a clear and structured way to evaluate content performance across different channels.  
This dashboard supports data-driven decision-making for content creators and analysts.

##  Objectives
- To analyze YouTube performance data effectively  
- To track key metrics such as views, likes, and comments  
- To measure audience engagement and interaction  
- To compare performance across multiple channels  
- To identify trends and patterns in content performance  
- To support better content strategy decisions

## Tools and Technologies
- Power BI  
- Power Query (Data Cleaning and Transformation)  
- DAX (Data Analysis Expressions)  
- Data Visualization Techniques  

## Dataset Description
The dataset includes detailed information about YouTube channels and videos.  
It contains attributes such as channel name, video title, views, likes, and comments.  
Additional fields include engagement rate and age of videos in days.  
The dataset is used to analyze performance trends and viewer interaction.  
It helps in comparing multiple channels and identifying top-performing content.  

## Key Metrics (KPIs)
- **Total Views:** 23 Billion  
- **Total Likes:** 238 Million  
- **Average Views:** 28 Million  
- **Total Channels:** 800  
- **Total Comments:** 2 Million  

These KPIs provide a quick overview of overall YouTube performance.  

## Data Analysis and Insights

### 1.Views Analysis
- The dashboard shows a total of 23 billion views across all channels.  
- It helps identify top-performing channels based on view count.  
- Higher views indicate better reach and audience engagement.  
- This analysis highlights popular content and trending creators.  
- It supports strategic decisions for content creation.  

### 2.Likes Analysis
- Likes represent audience appreciation for content.  
- Channels with higher likes indicate strong audience connection.  
- This metric helps evaluate content quality and popularity.  
- It also helps in comparing engagement across different channels.  
- Higher likes often correlate with better content performance.  

### 3.Comments Analysis
- Comments reflect audience interaction and feedback.  
- A total of 2 million comments shows active engagement.  
- This helps understand audience opinions and preferences.  
- Higher comments indicate more user involvement.  
- It supports improving content based on feedback.  

### 4.Engagement Rate Analysis
- Engagement rate measures how actively users interact with content.  
- It includes likes, comments, and views as key factors.  
- Higher engagement indicates better audience connection.  
- This metric helps evaluate content effectiveness.  
- It is useful for optimizing content strategies.  

### 5.Views Per Day Analysis
- This shows daily performance trends of videos and channels.  
- It helps identify consistent and high-performing creators.  
- Daily trends reveal growth patterns over time.  
- This analysis supports tracking content performance.  
- It helps in planning future content releases.  

###  6.Channel-wise Performance
- Different channels show different performance levels.  
- Some channels dominate in views, while others lead in engagement.  
- This comparison helps identify top creators.  
- It supports benchmarking across channels.  
- It helps in understanding competitive performance.  

##  Dashboard Features

- Multiple visualizations (Bar Chart, Pie Chart, Gauge, Line Chart)  
- Dynamic and responsive dashboard  
- User-friendly interface
